Language learning app.

Create a quiz from a book or any subject of interest e.g food greetings or just lesson 1. The words can be looked up in online dictionary. (yandex or mymemory.translated.net) or entered manually. Listen to the pronunciation of the words (from yandex and voicerss.org) You can also download a ready quiz from the cloud. In the Quiz tab you swipe to right if you know the answer. The current status of the quiz (remaining questions / total questions) is displayed at the top.

To create a quiz

  1. Press "+"
  2. Enter a name e.g "Animals" for the new quiz in the Tab "Home"
  3. Select Language pair e.g Swedish-Russian (sv-ru)
  4. Press "Create"
  5. Press "+" again to hide the create options
  6. Select tab with the "Edit" (pen) icon
  7. Enter a word e.g hund (dog) in this case pres button "sv-ru"
  8. Select a synonym if more feasible by touching it
  9. Press "Add"
  10. Look up in Wiktionary by press button WikiYou can copy from there and enter in the quiz as well
  11. Repeat until you have a convenient number of questions for a WordQuiz
  12. To run the Quiz select Quiz from the Games tab
  13. You can reherse Quiz in "Edit" tab
  14. If "i" is visable you have extra information to view it press "i"
  15. Go a head and swipe left , if you dont know the answer or right if you are confident until you pass the exam!
  16. If you like to practice your spelling press the keyboard icon the swiping is then locked until you get type the correct answer
  17. If you find some thing wrong go back to "Edit", the question should be selected, press the edit symbol and fix the error. You can also include extra information e.g word class

Games

    Quiz
    Hang Man. The well known hangman game with words from the selected quiz
    CrossWord. A crossword is generated from the selected quiz

Quiz Modes

  1. Normalquestion as text and swipe to right if you know the answer
  2. Text mode press the keyboard symbol. Then type the answer and swipe when you typed the right answer
  3. IllustratedPress image symbol to show the illustration if available
  4. Speech mode press the headphones. You listen to the questions

To retake a quiz after pass

  1. Press "Edit"
  2. Press "Reset"
  3. Green question means that it been answered
  4. Bold that extra information is available
  5. Press "Quiz" and you can repat the quiz
  6. Press "Reverse" and you can take the Quiz in reverse order

To Download an available Quiz

  1. Press download from cloud symbol
  2. Scroll around, a description off the quiz i shown at the top
  3. You can enter a search pattern by a key string and/or a optional languag key eg "sv" available quiz in swedish are listed
  4. Select a quiz with prefered language combination
  5. Press "Download"

To Upload a Quiz

  1. Press the upload to cloud symbol
  2. Add a short description
  3. Enter a key (password), this is needed if you want to remove the Quiz
  4. Press "Upload"

To Edit a Quiz

  1. Press "Edit" tab
  2. Select the question with the edit symbol
  3. If you are running a Quiz the current question is selected for edit just press Edit tab and do corrections needed
  4. To look upp detailed descripion of word by press buton Wiki
  5. Enter the new information
  6. You can fill the clipboard by first pressing and hold on the word and then press the copy symbol. Then you can search for more information in by press the search symbol in the upper left
  7. You can also change the individual status of word e.g uncheck it if you think you need to rehearse that particular word
  8. You can add an image representing the word by pressing the image symbol in the edit dialog
  9. Press Update to update the local database
  10. To update it on server folow the steps in "To Upload a Quiz" but press update at the end

To Delete an uploaded Quiz

  1. Press "Download"
  2. Select the quiz to remove
  3. Press "Remove"
  4. Enter the password entered when it was uploaded
