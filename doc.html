Language learning app.

Create your own quiz (Flashcard set) from a book or any subject of interest e.g food greetings. The words can be looked up in online dictionary or entered manually. Play games with the entered questions, listen to the pronunciation of the words rehearse and more.

To create a quiz (Flashcard set)

  1. Press "+"
  2. Enter a name e.g "Animals" for the new quiz in the Tab "Home"
  3. Select Language pair e.g Swedish-Russian (sv-ru)
  4. Press "Create"
  5. Press "+" again to hide the create options
  6. Select tab with the "Edit" (pen) icon
  7. Enter a word e.g hund (dog) in this case pres button "sv-ru"
  8. Select a synonym if more feasible by touching it
  9. Press "Add"
  10. Look up in Wiktionary by press button WikiYou can copy from there and enter in the quiz as well
  11. Repeat until you have a convenient number of questions for a WordQuiz
  12. To run the Quiz select Quiz from the Games tab
  13. You can reherse Quiz in "Edit" tab
  14. If "i" is visible you have extra information to view it press "i"
  15. Go a head and swipe left , if you dont know the answer or right if you are confident until you pass the exam!
  16. To view the answer press the flip symbol
  17. If you like to practice your spelling press the keyboard icon the swiping is then locked until you get type the correct answer
  18. If you find some thing wrong go back to "Edit", the question should be selected, press the edit symbol and fix the error. You can also include extra information e.g word class.

Games

  • Quiz (Flashcards). Answer questions turn around to se answer
  • Hang Man. The well known hangman game with words from the selected quiz
  • CrossWord. A crossword is generated from the selected quiz

Quiz Modes

  • Normalquestion as text and swipe to right if you know the answer or flip around to se the answer
  • Text mode press the keyboard symbol. Then type the answer and swipe when you typed the right answer
  • IllustratedPress image symbol to show the illustration if available
  • Speech mode press the headphones. You listen to the questions

To retake a quiz after pass

  1. Press "Edit"
  2. Press "Reset"
  3. Green question means that it been answered
  4. Bold that extra information is available
  5. Press "Quiz" and you can repat the quiz
  6. Press "Reverse" and you can take the Quiz in reverse order

To Download an available Quiz

  1. Press download from cloud symbol
  2. Scroll around, a description off the quiz i shown at the top
  3. You can enter a search pattern by a key string and/or a optional languag key eg "sv" available quiz in swedish are listed
  4. Select a quiz with prefered language combination
  5. Press "Download"

To Upload a Quiz

  1. Press the upload to cloud symbol
  2. Add a short description
  3. Enter a key (password), this is needed if you want to remove the Quiz
  4. Press "Upload"

To Edit a Quiz

  1. Press "Edit" tab
  2. Select the question with the edit symbol
  3. If you are running a Quiz the current question is selected for edit just press Edit tab and do corrections needed
  4. To look up detailed description of word by press button Wiki
  5. Enter the new information
  6. You can do search for information by the info in the clipboard by pressing the search symbol in the upper left.
  7. You can also change the individual status of word e.g uncheck it if you think you need to rehearse that particular word
  8. You can add an image representing the word by pressing the image symbol in the edit dialog. Copy the word and press search symbol and look for images download and go back and select image symbol an select the newly downloaded image
  9. Press Update to update the local database
  10. To update it on server folow the steps in "To Upload a Quiz" but press update at the end

To Delete an uploaded Quiz

  1. Press "Download"
  2. Select the quiz to remove
  3. Press "Remove"
  4. Enter the password entered when it was uploaded

Desktop version

  1. Install the windows version
  2. Go to https://github.com/faxling/WordQuizWin/releases/download/v2.0.0.8/WordQuiz_2.0.0.8.exe or the latest
  3. Install
  4. Download a Quiz
  5. Use it! Or create or update
  6. Take the Quiz and drag suitable images from e.g a browser To the app.If google image search Use "Open image in new tab"
  7. You can also drag the image to the Edit dialog
  8. Upload or update the Quiz to the "Cloud"
